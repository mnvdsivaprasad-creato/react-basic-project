# Task Manager App

A modern Task Manager application built with **React.js**, **Redux Toolkit**, and **Axios**. This project demonstrates real-world frontend development concepts including authentication, API integration, CRUD operations, routing, state management, and responsive task management features.

---

## Live Demo

**Vercel:** *http://react-basic-project-dfyi9q02i-mnvdsivaprasad-creatos-projects.vercel.app/dashboard*

---

## Features

- User Authentication
- Protected Routes
- LocalStorage Persistence
- Redux Toolkit State Management
- Redux Async Thunks
- Axios API Integration
- Create Task
- Update Task
- Delete Task
- View Task Details
- Search Tasks
- Filter Tasks by Status
- Sort Tasks (A–Z / Z–A)
- Pagination
- Dashboard Statistics
- Toast Notifications
- Loading and Error Handling

## Installation

Clone the repository:

```bash
git clone https://github.com/mnvdsivaprasad-creato/react-basic-project
```

Go to the project folder:

```bash
cd react-basic-project
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```
